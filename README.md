## 🎨 Admin Panel Features

<details open>
<summary><strong>🔐 Password Protection</strong></summary>
<br>

Default password: <code>admin123</code>  
Change it in <code>src/pages/admin/AdminPanel.jsx</code> — look for <code>ADMIN_PASSWORD</code>.

| Device | Behavior |
|--------|----------|
| **First visit** | Enter password → unlock for session |
| **Same browser** | Stays unlocked until tab closes |
| **New device** | Password prompt (works on every device) |

</details>

<br>

<details>
<summary><strong>🎵 Music Player</strong></summary>
<br>

Set a song in **Settings** tab. A floating player appears at bottom-right.

- Paste audio URL or **upload a file** from your device
- Supports MP3, streaming URLs, etc.
- YouTube not supported for audio
- Toggle on/off with the floating music button

</details>

<br>

<details>
<summary><strong>🖼️ Background Image / Video</strong></summary>
<br>

Set a background in **Settings** tab:

- **Image** — URL or upload from device
- **Video** — URL, upload, or **YouTube link** (auto-detected!)
- Toggle on/off with the eye button (bottom-left)

</details>

<br>

<details>
<summary><strong>🚀 Deploy to Production</strong></summary>
<br>

1. Generate a **GitHub classic token** with `repo` scope at  
   [`https://github.com/settings/tokens`](https://github.com/settings/tokens)

2. Go to `/admin` → **Settings** → paste token → **Save**

3. Click **Deploy to Production**

> Commits `public/data.json` to GitHub → Vercel auto-redeploys → all users see updates in ~30s.

</details>

<br>

<details>
<summary><strong>📂 Upload Files</strong></summary>
<br>

Every media setting supports **both**:

| Method | How it works |
|--------|-------------|
| **URL** | Paste a direct link (mp3, mp4, jpg, YouTube, etc.) |
| **Upload** | Pick a file from your device, stored as data URL in localStorage |

No external storage service needed.

</details>

## 📁 Project Layout

```
📦 Portfolio CMS
├── 📂 public/          → 📄 data.json (edit & deploy)
├── 📂 src/
│   ├── 📄 App.jsx      → Routes (/ + /admin)
│   ├── 📄 main.jsx     → Entry point
│   ├── 📂 lib/
│   │   └── 📄 store.js → Zustand (all CRUD)
│   ├── 📂 components/  → Hero, Skills, Projects, etc.
│   └── 📂 pages/
│       ├── 📂 public/  → Home, Layout
│       └── 📂 admin/
│           └── 📄 AdminPanel.jsx
```

## 🔒 Security

| Item | Storage |
|------|---------|
| **Admin password** | Hardcoded in `AdminPanel.jsx` (change before deploy) |
| **GitHub token** | Your browser's `localStorage` only |
| **Portfolio content** | `public/data.json` + `localStorage` |
| **Contact messages** | `localStorage` (per-browser) |
| **Uploaded media** | Data URLs in `localStorage` |
